# Simple AI Chatbot
This is a single-page web application. It allows users to chat with Google's Gemini AI. The project uses basic web technologies. It connects to the Gemini API via JavaScript.
## Features
Instant messaging. Users can send and receive messages in real-time.
Clean user interface (UI). The interface is simple and user-friendly. It is built with HTML.
Direct API integration. It uses the Fetch API to communicate with Google Gemini.
## Built With
1. HTML5. This provides the structure of the chat window.
2. JavaScript. This handles user input and API calls.
3. Gemini API. This is the "brain" of the chatbot
